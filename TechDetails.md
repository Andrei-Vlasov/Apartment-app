# Apartment 

## Why it was created
Our application helps people to find flat/house for rent/buy.

## Technical details
Backend:
- TypeScript
- Nest.js
- DTO (data-to-object)
- TypeORM + PostgreSQL
- Passport.js login/register
- bcrypt hashing

Frontend:
- JavaScript
- Next.js
- React.js
- Map API 

Linting:
- Husky 4
- ESLint
- Prettier
